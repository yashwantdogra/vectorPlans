# Vector Floor Plans

This repository contains 5 sample US residential floor plan PDFs designed for drywall estimation AI system testing.

## Files Included

1. **floor_plan_1_single_story_3br.pdf** - Single-Story 3BR Ranch House (1,800 sq ft, 9 ft ceiling)
2. **floor_plan_2_single_story_2br.pdf** - Single-Story 2BR Bungalow (1,200 sq ft, 8 ft ceiling)
3. **floor_plan_3_two_story_4br.pdf** - Two-Story 4BR Colonial (2,400 sq ft, 9 ft ceilings each floor)
4. **floor_plan_4_two_story_3br.pdf** - Two-Story 3BR Home (2,000 sq ft, 8 ft ceilings each floor)
5. **floor_plan_5_apartment_1br.pdf** - 1BR Apartment (800 sq ft, 8 ft ceiling)

## Features

- **Vector-based PDFs** - Scalable with precise measurements
- **Room labels and dimensions** - Clear identification of all spaces
- **Wall specifications** - Interior walls (5"), Exterior walls (6")
- **Ceiling heights** - Specified for accurate drywall calculations
- **Door and window openings** - Marked for estimation accuracy
- **Total square footage** - Noted for each floor plan

## Purpose

These floor plans are specifically designed to test AI systems for drywall estimation, providing realistic residential layouts with all necessary dimensions and specifications.

## Generation

All PDFs were generated using Python's ReportLab library, ensuring true vector format for maximum quality and scalability.
